# phuquoc-tools

푸꾸옥 여행 중 iOS/Android 브라우저에서 바로 쓰는 모바일 유틸 페이지.

## 포함 기능
- 푸꾸옥 / 서울 현재 시간 표시
- VND ↔ KRW 환율 계산
- 빠른 금액 버튼
- 자주 쓰는 베트남어 문장 카드
- 공항/ATM/결제/날씨 실전 팁

## 기술 방향
- 정적 HTML/CSS/JS
- 모바일 퍼스트
- iOS Safari / Android Chrome 대응
- 외부 빌드 도구 없이 바로 배포 가능

## 실행
브라우저에서 `index.html` 열기

## 배포
- Vercel: Static site
- Netlify: Drag & Drop
- GitHub Pages: 정적 호스팅 가능
